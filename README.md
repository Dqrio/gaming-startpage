# Gaming Purple Startpage (Mobile)

## Qué es
Una startpage estilo gamer con:
- Collage full screen (imágenes en /assets)
- Overlay morado + blur
- Noise (grain)
- Partículas sutiles
- Reloj en vivo
- Timer de sesión (desde que abriste la pestaña)
- Buscador (Google / YouTube)

## Cómo usar en Android (Chrome)
Chrome Android NO deja cambiar "Nueva pestaña", pero sí:
- Configuración -> Página principal -> Activar -> URL personalizada

Pega la URL de GitHub Pages cuando publiques.

## Importante
Debes reemplazar las imágenes en /assets por tus wallpapers reales.
Los nombres por defecto:
- cod.jpg
- warzone.jpg
- minecraft.jpg
- lol.jpg
- valorant.jpg
- gta.jpg
- apex.jpg
- fortnite.jpg

## Deploy en GitHub Pages
Settings -> Pages -> Deploy from branch -> main / root
